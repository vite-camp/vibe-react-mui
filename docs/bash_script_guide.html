‚ö†Ô∏èSystem Requirements

The script can CREATE everything from scratch, but to actually RUN the project you need:

  • Node.js (v16 or higher) - for running npm commands
  • npm (comes with Node.js) - for installing dependencies
  • Bash shell - for running the script

The magic: Even without Node.js installed, the script creates all files correctly. You only need Node.js when it tries to run npm install and npm run build.
